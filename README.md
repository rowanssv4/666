# 🚀 666 - 每日免费 VPN 节点自动收集器

一个基于 GitHub Actions 实现的自动化节点抓取与清洗工具。项目每 12 小时自动运行一次，从各大公开的高频优质源中提取并筛选节点，自动去重、测速过滤，并统一进行规范化重命名。

---

## ✨ 项目特性

* 🔄 **全自动化**：利用 GitHub Actions 定时触发，无需人工干预。
* 📅 **动态抓取**：智能适配日期变化的动态更新源，确保链接不失效。
* ⚡ **测速过滤**：内置 TCP 三次握手检测，自动剔除彻底失效的死节点，提升节点池整体质量。
* 🏷️ **智能重命名**：自动识别节点原始备注中的国家/地区（香港、台湾、日本、美国、韩国、新加坡等），并统一格式化为 `国家代码-Rowanss节点分享-编号`。
* 📢 **公告置顶**：首行自动注入特定的置顶声明节点（判定为不通），用于项目说明。
